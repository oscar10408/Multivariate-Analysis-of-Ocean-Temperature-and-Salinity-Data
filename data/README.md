# Processed data placeholder

This repository expects processed ARGO profile matrices in this directory, including:
- `lat.csv.gz`
- `lon.csv.gz`
- `date.csv.gz`
- `pressure.csv.gz`
- `temp.csv.gz`
- `psal.csv.gz`

Generate them by running one of the `get_data` scripts, then `prep.py` or `prep.jl`.
